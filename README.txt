【タイトル】
ギミック×変則盤面！遊び尽くせるオセロゲーム

【概要】
盤面サイズやギミックを自由に変更できるオセロゲームです。
通常のルールに加え、盤面の見た目や動作をカスタマイズ可能にし、飽きずに何度も遊べるように設計しています。

【主な特徴】
・盤面サイズ変更（6×6、8×8、10×10、12×8、17×8など）
・COM（ランダムAI）との対戦機能
・ヒント機能付きで置ける場所がわかりやすい
・石の枚数が一目でわかるゲージ表示
・Specialモードでは30種類以上の特殊盤面で遊べます

【特殊ギミックの例】
・穴マス（置けない・ひっくり返らない）
・ループ盤面（上下左右でつながっている）
・拡大／縮小マス（見た目の変化）
・マスごとの重みづけ（得点差のある戦い）
・反転方向変更（斜め、回転など）

【使用技術】
・JavaScript（p5.js）
・HTML/CSS
・クラス設計による盤面の構造化

【今後の改善点】
・効果音・BGMの追加
・アニメーションによる演出強化
・外部ファイルで盤面情報を管理可能に
・リプレイ機能の実装

【操作方法】
ブラウザで index.html を開くだけでプレイ可能です。
操作環境はPCを推奨します。
